Bem-vindo à API de Cidades Inteligentes!

A API de Cidades Inteligentes foi desenvolvida para integrar e fornecer dados relevantes sobre a infraestrutura urbana, segurança e meio ambiente, ajudando a promover uma gestão pública mais eficaz e a construir cidades mais sustentáveis e inteligentes.

Funcionalidades

Nosso sistema permite o acesso a dados em tempo real e análises sobre:

  • Dados de Tráfego: Monitora o fluxo de veículos, congestionamentos e informações sobre as principais vias da cidade.
  • Sensor Ambiental: Coleta dados de temperatura, umidade e qualidade do ar em diferentes locais, contribuindo para a saúde e bem-estar da população.

Endpoints da API

Utilize os seguintes endpoints para acessar os dados disponíveis:

  • Dados de Tráfego - Obtenha informações sobre o tráfego atual, incluindo número de veículos e condições de congestionamento.
  • Sensor Ambiental - Acesse dados coletados por sensores sobre temperatura, umidade e qualidade do ar.

Documentação da API

Para mais detalhes sobre como utilizar nossa API, consulte a documentação completa disponível em Swagger. Esta documentação inclui exemplos de requisições e respostas, bem como informações sobre autenticação e uso de endpoints.
